# 🚀 DevTrend — Developer News & Tool Explorer

## 📌 Overview

DevTrend is a web application that allows developers to explore trending articles, tools, and topics in the tech ecosystem. It fetches real-time data from the Dev.to API and provides a clean and interactive interface for users.

## 🎯 Features (Planned)

* Search articles by keywords
* Filter articles based on tags (e.g., AI, Web Development, Python)
* Sort articles by date or popularity
* Responsive design for mobile, tablet, and desktop
* Dark/Light mode toggle (optional)
* Save favorite articles (optional)

## 🌐 API Used

* Dev.to API
  Endpoint: https://dev.to/api/articles

## 🛠️ Technologies Used

* HTML
* CSS (or Tailwind/Bootstrap optional)
* JavaScript (ES6+)
* Fetch API

## ⚙️ How to Run

1. Clone the repository
2. Open index.html in your browser
